* About
This repository is for storing my /tmux/ configuration files. As usual, I'll try to make this configuration as vanilla as possible.

* Configuration
:PROPERTIES:

** General Configurations
*** Use /tmux-256color/ if possible
#+begin_src conf
  if 'infocmp -x tmux-256color > /dev/null 2>&1' 'set -g default-terminal "tmux-256color"'
#+end_src
*** Screen keybindings
#+begin_src conf
  unbind C-b
  set -g prefix C-a
  bind a send-prefix
  bind a send-prefix
  bind-key C-a last-window
#+end_src
I'm already too familiar with /GNU Screen/, so ~C-a~ is always my prefix key. To make it more /Screen-like/, ~C-a C-a~ will alternate between two latest windows.
*** xterm-keys
#+begin_src conf
  setw -g xterm-keys on
#+end_src
*** Automatic Rename
Rename the window to reflect the current program.
#+begin_src conf
  setw -g automatic-rename on
#+end_src
*** Window Renumbering
Renumber windows when a window is closed.
#+begin_src conf
  set -g renumber-windows on
#+end_src
*** Set the terminal title
#+begin_src conf
  set -g set-titles on
#+end_src
*** Monitor activity
Monitor for activity in the window. Windows with activity are highlighted in the status line.
#+begin_src conf
  set -g monitor-activity on
#+end_src
*** Visual activity
If on, display a message instead of sending a bell when activity occurs in a window for which the monitor-activity window option is enabled. If set to both, a bell and a message are produced.
#+begin_src conf
  set -g visual-activity on
#+end_src

*** Clear both screen and history
#+begin_src conf
  bind -n C-l send-keys C-l \; run 'sleep 0.2' \; clear-history
#+end_src
*** New Session
Create a new session,
#+begin_src conf
  bind C-c new-session
#+end_src

#+begin_src conf
  bind C-f command-prompt -p find-session 'switch-client -t %%'

#+begin_src conf
  bind e new-window -n "~/.config/tmux/tmux.conf" sh -c '${EDITOR:-vim} ~/.config/tmux/tmux.conf && tmux source ~/.tmux.conf && tmux display "~/.config/tmux/tmux.conf sourced"'
#+end_src
